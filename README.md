# Stepwise – Walk the World 🌍

Stepwise is an Android app that turns your daily steps into progress along real-world routes.
Instead of just counting steps, you walk through curated tracks made up of meaningful milestones. Every step moves you forward on your selected route.

### 🗺 Select Tracks

Choose from different virtual tracks inspired by real-world routes and locations.

### 📍 Milestones

Each track consists of multiple milestones positioned at specific distances.
When you reach a milestone, the app shows:

- Information about the location

- Photos and interesting facts

- A progress overview of your journey

### 👣 Step-Based Progress

Your progress is calculated using:

```distance = steps × step_length```

Your real-world activity translates directly into virtual movement.

### 🏆 Achievements

Unlock achievements based on:

- Total distance walked

- Completed tracks

- Reached milestones

### 🔗 Fitbit Integration

The app can optionally connect to the Fitbit Web API so users can sync their Fitbit step data (work in progress).

Each user authenticates with their own Fitbit account via OAuth 2.0.
No Fitbit credentials are stored by this project.

### 📱 Platform

Android (Java)

Uses device step counter and optional Fitbit API integration

### 🎯 Purpose

This project is designed to make walking more engaging by turning everyday movement into an interactive journey.
